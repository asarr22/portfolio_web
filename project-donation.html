Overview

This platform lets an organization run donation campaigns through a Flutter app, while a separate administrator application handles campaign creation, monitoring, and reporting.

The backend runs on Firebase, handling authentication, campaign data, and donation records in real time so the admin dashboard reflects activity as it happens.

Design priorities were clarity and trust — donors needed to see exactly where a campaign stood, and admins needed a dashboard simple enough to run without training.

Highlights

  • Separate donor-facing and administrator Flutter apps
  • Campaign creation, tracking, and reporting dashboard
  • Firebase-backed authentication and real-time data sync
  • Cloud backend integration for donation records
